# Definition Of Done

一个功能只有同时满足以下条件，才视为完成。

## 规格完成

- 对应 `proposal/spec/tasks/acceptance` 已创建并补全必要字段
- 需求范围、非目标、异常流程明确
- 验收标准清晰且可执行

## 实现完成

- 代码与 `spec` 一致
- 关键状态齐全：加载、空、错误、权限不足
- 日志、兜底和重试机制满足项目规范

## 质量完成

- TypeScript 严格模式通过
- ESLint 通过
- Prettier 通过
- 关键测试和 harness 通过

## 验收完成

- UI 验收完成或明确说明不适用
- 问题清单已关闭或记录了剩余风险
- 验收结论已写入 `acceptance.md`

## 可追踪完成

- 功能关联的任务 ID、分支、提交或 PR 可追踪
- 关键验证证据可定位
