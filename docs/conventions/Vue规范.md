# Vue 规范

## 命名规范

- 组件`export`的外部代码，应视为全局代码，变量命名应为<b>所有字母大写与下划线</b>。
- 优先使用`const`关键字声明变量。
- 不使用`var`关键字。
- `data`中命名，应视为局部代码，变量命名应为<b>所有字母小写与下划线</b>。
- 默认使用双引号，双引号冲突的情况下，使用单引号。
- 自定义事件使用`小写-横线`命名。
- `v-bind`和`v-on`使用简写。
- `props`的定义，必须包含`type, required`。
- 使用自定义组件使用**横线连接 (kebab-case)**形式

## 语句

- `method`方法中，<b>每个函数之间保留一行空行</b>。

## 样式

- 所有非顶级组件，`style`必须添加`scoped`。
- 满足`CSS`规范。
