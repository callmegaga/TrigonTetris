# 编码规范

该目录包含了项目中使用的编码规范和最佳实践。遵循这些规范有助于提高代码的可读性、可维护性和一致性。请确保在编写代码时参考这些文档，以保持项目的高质量标准。

## 快速导航

- [JS和TS规范](JS和TS规范.md): 涵盖了 JavaScript 和 TypeScript 的编码规范，包括变量命名、函数定义、类型注解等方面的最佳实践。
- [CSS规范](CSS规范.md): 提供了 CSS 编码规范，涵盖了选择器命名、属性排序、缩进等方面的最佳实践。
- [HTML规范](HTML规范.md): 包含了 HTML 编码规范，涉及标签使用、属性命名、文档结构等方面的最佳实践。
- [Vue 规范](Vue规范.md): 介绍了 Vue.js 编码规范，包括组件命名、事件命名、样式规范等方面的最佳实践。
